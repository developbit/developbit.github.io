State Of The Web Data from Dev.to


As mentioned in a post from Dev.to, there was a survey that was released a few weeks ago to learn more about the State of the Web. In their article, they shared 43 survey questions in the form of an Excel or CSV file. Dev.to mentioned if we like to share our insights, a comment can be provided or publish the insights by August 29, 2018. Here are my insights:

My insights are broken down into five categories which include: Browser, Software Developer Behavior, Animal Preference, A/B Testing, and State of the Web Scale Questions.

Browser

The browser section displays: the primary count for browser development, technologies made use of during development, the age of apps, internet connection speed, and the operating system used for development.



Software Development Behavior

The software development behavior section displays: software development roles, front-end Javascript Frameworks, CSS tools to use, Years as a Software Developer, Age, Gender, Educational Background, and Personal Mobile Usage.


Testing

The Testing section displays: how A/B testing is done within a project and A/B Testing Questions.


Animal Preference

The Animal Preference section displays: what type of animal person are you.


State of the Web Questions

The State of the Web Questions section displays alist of questions ranging on the scale of 1 to 10.
